How to Run

terminal
$ python doe.py info --config use_cases/22_prefetch_strategy/config.json $ python doe.py generate --config use_cases/22_prefetch_strategy/config.json --output results/run.sh --seed 42 $ bash results/run.sh $ python doe.py analyze --config use_cases/22_prefetch_strategy/config.json $ python doe.py optimize --config use_cases/22_prefetch_strategy/config.json $ python doe.py report --config use_cases/22_prefetch_strategy/config.json --output report.html
