Box-Behnken Design

LED Strip Installation

Box-Behnken design to maximize brightness uniformity and minimize hot spots by tuning strip density, power supply headroom, and diffuser distance

Summary

This experiment investigates led strip installation. Box-Behnken design to maximize brightness uniformity and minimize hot spots by tuning strip density, power supply headroom, and diffuser distance.

The design varies 3 factors: leds per m (LEDs/m), ranging from 30 to 120, psu headroom pct (%), ranging from 10 to 40, and diffuser mm (mm), ranging from 5 to 30. The goal is to optimize 2 responses: uniformity pct (%) (maximize) and hotspot temp c (C) (minimize). Fixed conditions held constant across all runs include voltage = 12V, color = warm_white.

A Box-Behnken design was chosen because it efficiently fits quadratic models with 3 continuous factors while avoiding extreme corner combinations — requiring only 15 runs instead of the 8 needed for a full factorial at two levels.

Quadratic response surface models were fitted to capture potential curvature and factor interactions. The RSM contour plots below visualize how pairs of factors jointly affect each response.

Key Findings

For uniformity pct, the most influential factors were diffuser mm (35.7%), leds per m (35.3%), psu headroom pct (29.1%). The best observed value was 88.0 (at leds per m = 75, psu headroom pct = 10, diffuser mm = 30).

For hotspot temp c, the most influential factors were leds per m (51.2%), diffuser mm (29.3%), psu headroom pct (19.5%). The best observed value was 24.0 (at leds per m = 120, psu headroom pct = 40, diffuser mm = 17.5).

Recommended Next Steps

  • Run confirmation experiments at the predicted optimal settings to validate the model.
  • Consider whether any fixed factors should be varied in a future study.

Step-by-Step Workflow

1

Preview the design

Terminal
$ python doe.py info --config use_cases/271_led_strip_install/config.json
2

Generate the runner script

Terminal
$ python doe.py generate --config use_cases/271_led_strip_install/config.json \ --output use_cases/271_led_strip_install/results/run.sh --seed 42
3

Execute the experiments

Terminal
$ bash use_cases/271_led_strip_install/results/run.sh
4

Analyze results

Terminal
$ python doe.py analyze --config use_cases/271_led_strip_install/config.json
5

Get optimization recommendations

Terminal
$ python doe.py optimize --config use_cases/271_led_strip_install/config.json
6

Multi-objective optimization

With 2 competing responses, use --multi to find the best compromise via Derringer–Suich desirability.

Terminal
$ python doe.py optimize --config use_cases/271_led_strip_install/config.json --multi
7

Generate the HTML report

Terminal
$ python doe.py report --config use_cases/271_led_strip_install/config.json \ --output use_cases/271_led_strip_install/results/report.html
