Box-Behnken Design

UV-Vis Spectrophotometry

Box-Behnken design to maximize signal-to-noise ratio and minimize baseline drift by tuning slit width, scan speed, and sample path length

Summary

This experiment investigates uv-vis spectrophotometry. Box-Behnken design to maximize signal-to-noise ratio and minimize baseline drift by tuning slit width, scan speed, and sample path length.

The design varies 3 factors: slit nm (nm), ranging from 0.5 to 5.0, scan speed (nm/min), ranging from 100 to 1000, and path cm (cm), ranging from 0.1 to 5.0. The goal is to optimize 2 responses: snr (ratio) (maximize) and baseline drift (AU/hr) (minimize). Fixed conditions held constant across all runs include lamp = deuterium, wavelength = 260nm.

A Box-Behnken design was chosen because it efficiently fits quadratic models with 3 continuous factors while avoiding extreme corner combinations — requiring only 15 runs instead of the 8 needed for a full factorial at two levels.

Quadratic response surface models were fitted to capture potential curvature and factor interactions. The RSM contour plots below visualize how pairs of factors jointly affect each response.

Key Findings

For snr, the most influential factors were scan speed (37.4%), slit nm (32.9%), path cm (29.7%). The best observed value was 286.0 (at slit nm = 2.75, scan speed = 1000, path cm = 5).

For baseline drift, the most influential factors were scan speed (42.8%), path cm (31.7%), slit nm (25.5%). The best observed value was 0.0025 (at slit nm = 0.5, scan speed = 1000, path cm = 2.55).

Recommended Next Steps

  • Run confirmation experiments at the predicted optimal settings to validate the model.
  • Consider whether any fixed factors should be varied in a future study.

Step-by-Step Workflow

1

Preview the design

Terminal
$ python doe.py info --config use_cases/193_spectrophotometry/config.json
2

Generate the runner script

Terminal
$ python doe.py generate --config use_cases/193_spectrophotometry/config.json \ --output use_cases/193_spectrophotometry/results/run.sh --seed 42
3

Execute the experiments

Terminal
$ bash use_cases/193_spectrophotometry/results/run.sh
4

Analyze results

Terminal
$ python doe.py analyze --config use_cases/193_spectrophotometry/config.json
5

Get optimization recommendations

Terminal
$ python doe.py optimize --config use_cases/193_spectrophotometry/config.json
6

Multi-objective optimization

With 2 competing responses, use --multi to find the best compromise via Derringer–Suich desirability.

Terminal
$ python doe.py optimize --config use_cases/193_spectrophotometry/config.json --multi
7

Generate the HTML report

Terminal
$ python doe.py report --config use_cases/193_spectrophotometry/config.json \ --output use_cases/193_spectrophotometry/results/report.html
