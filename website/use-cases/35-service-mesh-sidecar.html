Step-by-Step Workflow

1

Preview the design

Terminal
$ python doe.py info --config use_cases/35_service_mesh_sidecar/config.json
2

Generate the runner script

Terminal
$ python doe.py generate --config use_cases/35_service_mesh_sidecar/config.json \ --output use_cases/35_service_mesh_sidecar/results/run.sh --seed 42
3

Execute the experiments

Terminal
$ bash use_cases/35_service_mesh_sidecar/results/run.sh
4

Analyze results

Terminal
$ python doe.py analyze --config use_cases/35_service_mesh_sidecar/config.json
5

Get optimization recommendations

Terminal
$ python doe.py optimize --config use_cases/35_service_mesh_sidecar/config.json
6

Generate the HTML report

Terminal
$ python doe.py report --config use_cases/35_service_mesh_sidecar/config.json \ --output use_cases/35_service_mesh_sidecar/results/report.html
