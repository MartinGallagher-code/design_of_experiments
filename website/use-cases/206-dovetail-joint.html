Box-Behnken Design

Dovetail Joint Aesthetics

Box-Behnken design to maximize visual appeal and joint tightness by tuning tail angle, pin-to-tail ratio, and baseline offset

Summary

This experiment investigates dovetail joint aesthetics. Box-Behnken design to maximize visual appeal and joint tightness by tuning tail angle, pin-to-tail ratio, and baseline offset.

The design varies 3 factors: tail angle deg (deg), ranging from 7 to 14, pin tail ratio (ratio), ranging from 0.3 to 0.8, and baseline mm (mm), ranging from 0.5 to 2.0. The goal is to optimize 2 responses: visual appeal (pts) (maximize) and tightness (pts) (maximize). Fixed conditions held constant across all runs include wood = cherry, board thickness = 19mm.

A Box-Behnken design was chosen because it efficiently fits quadratic models with 3 continuous factors while avoiding extreme corner combinations — requiring only 15 runs instead of the 8 needed for a full factorial at two levels.

Quadratic response surface models were fitted to capture potential curvature and factor interactions. The RSM contour plots below visualize how pairs of factors jointly affect each response.

Key Findings

For visual appeal, the most influential factors were pin tail ratio (39.8%), baseline mm (36.2%), tail angle deg (24.1%). The best observed value was 7.3 (at tail angle deg = 7, pin tail ratio = 0.3, baseline mm = 1.25).

For tightness, the most influential factors were pin tail ratio (47.1%), tail angle deg (27.3%), baseline mm (25.6%). The best observed value was 7.4 (at tail angle deg = 10.5, pin tail ratio = 0.3, baseline mm = 2).

Recommended Next Steps

  • Run confirmation experiments at the predicted optimal settings to validate the model.
  • Consider whether any fixed factors should be varied in a future study.

Step-by-Step Workflow

1

Preview the design

Terminal
$ python doe.py info --config use_cases/206_dovetail_joint/config.json
2

Generate the runner script

Terminal
$ python doe.py generate --config use_cases/206_dovetail_joint/config.json \ --output use_cases/206_dovetail_joint/results/run.sh --seed 42
3

Execute the experiments

Terminal
$ bash use_cases/206_dovetail_joint/results/run.sh
4

Analyze results

Terminal
$ python doe.py analyze --config use_cases/206_dovetail_joint/config.json
5

Get optimization recommendations

Terminal
$ python doe.py optimize --config use_cases/206_dovetail_joint/config.json
6

Multi-objective optimization

With 2 competing responses, use --multi to find the best compromise via Derringer–Suich desirability.

Terminal
$ python doe.py optimize --config use_cases/206_dovetail_joint/config.json --multi
7

Generate the HTML report

Terminal
$ python doe.py report --config use_cases/206_dovetail_joint/config.json \ --output use_cases/206_dovetail_joint/results/report.html
