Box-Behnken Design

Film Development Process

Box-Behnken design to maximize tonal range and minimize grain by tuning developer temperature, agitation frequency, and development time

Summary

This experiment investigates film development process. Box-Behnken design to maximize tonal range and minimize grain by tuning developer temperature, agitation frequency, and development time.

The design varies 3 factors: dev temp c (C), ranging from 18 to 24, agitation per min (inversions/min), ranging from 1 to 6, and dev time min (min), ranging from 6 to 14. The goal is to optimize 2 responses: tonal range (zones) (maximize) and grain score (pts) (minimize). Fixed conditions held constant across all runs include film type = ilford_hp5, developer = rodinal.

A Box-Behnken design was chosen because it efficiently fits quadratic models with 3 continuous factors while avoiding extreme corner combinations — requiring only 15 runs instead of the 8 needed for a full factorial at two levels.

Quadratic response surface models were fitted to capture potential curvature and factor interactions. The RSM contour plots below visualize how pairs of factors jointly affect each response.

Key Findings

For tonal range, the most influential factors were agitation per min (35.1%), dev temp c (32.9%), dev time min (32.0%). The best observed value was 7.8 (at dev temp c = 21, agitation per min = 3.5, dev time min = 10).

For grain score, the most influential factors were agitation per min (47.0%), dev time min (36.7%), dev temp c (16.3%). The best observed value was 2.9 (at dev temp c = 21, agitation per min = 3.5, dev time min = 10).

Recommended Next Steps

  • Run confirmation experiments at the predicted optimal settings to validate the model.
  • Consider whether any fixed factors should be varied in a future study.

Step-by-Step Workflow

1

Preview the design

Terminal
$ python doe.py info --config use_cases/152_film_development/config.json
2

Generate the runner script

Terminal
$ python doe.py generate --config use_cases/152_film_development/config.json \ --output use_cases/152_film_development/results/run.sh --seed 42
3

Execute the experiments

Terminal
$ bash use_cases/152_film_development/results/run.sh
4

Analyze results

Terminal
$ python doe.py analyze --config use_cases/152_film_development/config.json
5

Get optimization recommendations

Terminal
$ python doe.py optimize --config use_cases/152_film_development/config.json
6

Multi-objective optimization

With 2 competing responses, use --multi to find the best compromise via Derringer–Suich desirability.

Terminal
$ python doe.py optimize --config use_cases/152_film_development/config.json --multi
7

Generate the HTML report

Terminal
$ python doe.py report --config use_cases/152_film_development/config.json \ --output use_cases/152_film_development/results/report.html
