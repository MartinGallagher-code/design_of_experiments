Headlight Beam Alignment

Full factorial of vertical aim, horizontal aim, bulb wattage, and lens clarity to maximize road illumination and minimize oncoming glare

Summary

This experiment investigates headlight beam alignment. Full factorial of vertical aim, horizontal aim, bulb wattage, and lens clarity to maximize road illumination and minimize oncoming glare.

The design varies 4 factors: vertical deg (deg), ranging from -2 to 2, horizontal deg (deg), ranging from -1 to 1, bulb watts (W), ranging from 35 to 65, and lens clarity (%), ranging from 50 to 100. The goal is to optimize 2 responses: illumination lux (lux) (maximize) and glare score (pts) (minimize). Fixed conditions held constant across all runs include headlight type = reflector, beam = low.

A full factorial design was used to explore all 16 possible combinations of the 4 factors at two levels. This guarantees that every main effect and interaction can be estimated independently, at the cost of a larger experiment (16 runs).

Quadratic response surface models were fitted to capture potential curvature and factor interactions. The RSM contour plots below visualize how pairs of factors jointly affect each response.

Key Findings

For illumination lux, the most influential factors were vertical deg (50.0%), horizontal deg (26.4%), lens clarity (16.4%). The best observed value was 210.0 (at vertical deg = 2, horizontal deg = 1, bulb watts = 65).

For glare score, the most influential factors were vertical deg (36.1%), bulb watts (28.6%), lens clarity (26.3%). The best observed value was 1.0 (at vertical deg = 2, horizontal deg = -1, bulb watts = 65).

Recommended Next Steps

  • Consider whether any fixed factors should be varied in a future study.

Step-by-Step Workflow

1

Preview the design

Terminal
$ python doe.py info --config use_cases/124_headlight_alignment/config.json
2

Generate the runner script

Terminal
$ python doe.py generate --config use_cases/124_headlight_alignment/config.json \ --output use_cases/124_headlight_alignment/results/run.sh --seed 42
3

Execute the experiments

Terminal
$ bash use_cases/124_headlight_alignment/results/run.sh
4

Analyze results

Terminal
$ python doe.py analyze --config use_cases/124_headlight_alignment/config.json
5

Get optimization recommendations

Terminal
$ python doe.py optimize --config use_cases/124_headlight_alignment/config.json
6

Multi-objective optimization

With 2 competing responses, use --multi to find the best compromise via Derringer–Suich desirability.

Terminal
$ python doe.py optimize --config use_cases/124_headlight_alignment/config.json --multi
7

Generate the HTML report

Terminal
$ python doe.py report --config use_cases/124_headlight_alignment/config.json \ --output use_cases/124_headlight_alignment/results/report.html
