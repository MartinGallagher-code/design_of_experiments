Box-Behnken Design

Raised Bed Soil Mix

Box-Behnken design to optimize drainage and root growth by tuning peat moss ratio, perlite ratio, and compost ratio in a raised bed mix

Summary

This experiment investigates raised bed soil mix. Box-Behnken design to optimize drainage and root growth by tuning peat moss ratio, perlite ratio, and compost ratio in a raised bed mix.

The design varies 3 factors: peat pct (%), ranging from 20 to 50, perlite pct (%), ranging from 10 to 35, and compost pct (%), ranging from 15 to 45. The goal is to optimize 2 responses: drainage rate (mm/hr) (maximize) and root growth (cm/week) (maximize). Fixed conditions held constant across all runs include bed depth cm = 30, remaining topsoil = balance.

A Box-Behnken design was chosen because it efficiently fits quadratic models with 3 continuous factors while avoiding extreme corner combinations — requiring only 15 runs instead of the 8 needed for a full factorial at two levels.

Quadratic response surface models were fitted to capture potential curvature and factor interactions. The RSM contour plots below visualize how pairs of factors jointly affect each response.

Key Findings

For drainage rate, the most influential factors were perlite pct (59.4%), peat pct (26.2%), compost pct (14.4%). The best observed value was 65.0 (at peat pct = 20, perlite pct = 35, compost pct = 30).

For root growth, the most influential factors were compost pct (57.0%), peat pct (24.6%), perlite pct (18.3%). The best observed value was 4.16 (at peat pct = 50, perlite pct = 10, compost pct = 30).

Recommended Next Steps

  • Run confirmation experiments at the predicted optimal settings to validate the model.
  • Consider whether any fixed factors should be varied in a future study.

Step-by-Step Workflow

1

Preview the design

Terminal
$ python doe.py info --config use_cases/106_raised_bed_mix/config.json
2

Generate the runner script

Terminal
$ python doe.py generate --config use_cases/106_raised_bed_mix/config.json \ --output use_cases/106_raised_bed_mix/results/run.sh --seed 42
3

Execute the experiments

Terminal
$ bash use_cases/106_raised_bed_mix/results/run.sh
4

Analyze results

Terminal
$ python doe.py analyze --config use_cases/106_raised_bed_mix/config.json
5

Get optimization recommendations

Terminal
$ python doe.py optimize --config use_cases/106_raised_bed_mix/config.json
6

Multi-objective optimization

With 2 competing responses, use --multi to find the best compromise via Derringer–Suich desirability.

Terminal
$ python doe.py optimize --config use_cases/106_raised_bed_mix/config.json --multi
7

Generate the HTML report

Terminal
$ python doe.py report --config use_cases/106_raised_bed_mix/config.json \ --output use_cases/106_raised_bed_mix/results/report.html
