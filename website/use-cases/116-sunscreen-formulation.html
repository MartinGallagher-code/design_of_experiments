Central Composite Design

Sunscreen SPF Formulation

Central composite design to maximize SPF protection and minimize greasiness by tuning zinc oxide concentration, moisturizer ratio, and application thickness

Summary

This experiment investigates sunscreen spf formulation. Central composite design to maximize SPF protection and minimize greasiness by tuning zinc oxide concentration, moisturizer ratio, and application thickness.

The design varies 3 factors: zinc oxide pct (%), ranging from 5 to 25, moisturizer pct (%), ranging from 10 to 40, and thickness mg cm2 (mg/cm2), ranging from 1.0 to 2.5. The goal is to optimize 2 responses: spf value (SPF) (maximize) and greasiness (pts) (minimize). Fixed conditions held constant across all runs include base type = cream, uva filter = avobenzone.

A Central Composite Design (CCD) was selected to fit a full quadratic response surface model, including curvature and interaction effects. With 3 factors this produces 22 runs including center points and axial (star) points that extend beyond the factorial range.

Quadratic response surface models were fitted to capture potential curvature and factor interactions. The RSM contour plots below visualize how pairs of factors jointly affect each response.

Key Findings

For spf value, the most influential factors were zinc oxide pct (42.4%), thickness mg cm2 (42.4%), moisturizer pct (15.3%). The best observed value was 52.0 (at zinc oxide pct = 15, moisturizer pct = 25, thickness mg cm2 = 0.380694).

For greasiness, the most influential factors were thickness mg cm2 (48.0%), zinc oxide pct (39.8%), moisturizer pct (12.2%). The best observed value was 1.3 (at zinc oxide pct = 15, moisturizer pct = 25, thickness mg cm2 = 3.11931).

Recommended Next Steps

  • Run confirmation experiments at the predicted optimal settings to validate the model.
  • Consider whether any fixed factors should be varied in a future study.

Step-by-Step Workflow

1

Preview the design

Terminal
$ python doe.py info --config use_cases/116_sunscreen_formulation/config.json
2

Generate the runner script

Terminal
$ python doe.py generate --config use_cases/116_sunscreen_formulation/config.json \ --output use_cases/116_sunscreen_formulation/results/run.sh --seed 42
3

Execute the experiments

Terminal
$ bash use_cases/116_sunscreen_formulation/results/run.sh
4

Analyze results

Terminal
$ python doe.py analyze --config use_cases/116_sunscreen_formulation/config.json
5

Get optimization recommendations

Terminal
$ python doe.py optimize --config use_cases/116_sunscreen_formulation/config.json
6

Multi-objective optimization

With 2 competing responses, use --multi to find the best compromise via Derringer–Suich desirability.

Terminal
$ python doe.py optimize --config use_cases/116_sunscreen_formulation/config.json --multi
7

Generate the HTML report

Terminal
$ python doe.py report --config use_cases/116_sunscreen_formulation/config.json \ --output use_cases/116_sunscreen_formulation/results/report.html
