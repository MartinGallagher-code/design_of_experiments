Box-Behnken Design

Photo Print Color Accuracy

Box-Behnken design to maximize color accuracy and minimize ink cost by tuning color profile, ink density, and paper type brightness

Summary

This experiment investigates photo print color accuracy. Box-Behnken design to maximize color accuracy and minimize ink cost by tuning color profile, ink density, and paper type brightness.

The design varies 3 factors: profile gamma (gamma), ranging from 1.8 to 2.4, ink density pct (%), ranging from 80 to 120, and paper brightness (ISO), ranging from 90 to 100. The goal is to optimize 2 responses: delta e (dE) (minimize) and ink ml (mL/m2) (minimize). Fixed conditions held constant across all runs include printer = inkjet_8color, resolution dpi = 1440.

A Box-Behnken design was chosen because it efficiently fits quadratic models with 3 continuous factors while avoiding extreme corner combinations — requiring only 15 runs instead of the 8 needed for a full factorial at two levels.

Quadratic response surface models were fitted to capture potential curvature and factor interactions. The RSM contour plots below visualize how pairs of factors jointly affect each response.

Key Findings

For delta e, the most influential factors were profile gamma (43.9%), paper brightness (42.7%), ink density pct (13.5%). The best observed value was 3.8 (at profile gamma = 2.4, ink density pct = 100, paper brightness = 90).

For ink ml, the most influential factors were ink density pct (41.8%), paper brightness (30.2%), profile gamma (28.0%). The best observed value was 5.6 (at profile gamma = 2.4, ink density pct = 120, paper brightness = 95).

Recommended Next Steps

  • Run confirmation experiments at the predicted optimal settings to validate the model.
  • Consider whether any fixed factors should be varied in a future study.

Step-by-Step Workflow

1

Preview the design

Terminal
$ python doe.py info --config use_cases/155_photo_print_color/config.json
2

Generate the runner script

Terminal
$ python doe.py generate --config use_cases/155_photo_print_color/config.json \ --output use_cases/155_photo_print_color/results/run.sh --seed 42
3

Execute the experiments

Terminal
$ bash use_cases/155_photo_print_color/results/run.sh
4

Analyze results

Terminal
$ python doe.py analyze --config use_cases/155_photo_print_color/config.json
5

Get optimization recommendations

Terminal
$ python doe.py optimize --config use_cases/155_photo_print_color/config.json
6

Multi-objective optimization

With 2 competing responses, use --multi to find the best compromise via Derringer–Suich desirability.

Terminal
$ python doe.py optimize --config use_cases/155_photo_print_color/config.json --multi
7

Generate the HTML report

Terminal
$ python doe.py report --config use_cases/155_photo_print_color/config.json \ --output use_cases/155_photo_print_color/results/report.html
