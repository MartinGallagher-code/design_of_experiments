Oyster Reef Substrate Design

Box-Behnken design to maximize oyster settlement and reef height by tuning substrate rugosity, shell depth, and placement elevation

Summary

This experiment investigates oyster reef substrate design. Box-Behnken design to maximize oyster settlement and reef height by tuning substrate rugosity, shell depth, and placement elevation.

The design varies 3 factors: rugosity cm (cm), ranging from 2 to 10, shell depth cm (cm), ranging from 10 to 40, and elevation m (m_MSL), ranging from -1 to 1. The goal is to optimize 2 responses: settlement per m2 (spat/m2) (maximize) and reef height cm (cm/yr) (maximize). Fixed conditions held constant across all runs include species = crassostrea_virginica, site = estuary_channel.

A Box-Behnken design was chosen because it efficiently fits quadratic models with 3 continuous factors while avoiding extreme corner combinations — requiring only 15 runs instead of the 8 needed for a full factorial at two levels.

Quadratic response surface models were fitted to capture potential curvature and factor interactions. The RSM contour plots below visualize how pairs of factors jointly affect each response.

Key Findings

For settlement per m2, the most influential factors were elevation m (37.0%), shell depth cm (33.9%), rugosity cm (29.1%). The best observed value was 189.0 (at rugosity cm = 6, shell depth cm = 25, elevation m = 0).

For reef height cm, the most influential factors were elevation m (42.9%), shell depth cm (36.1%), rugosity cm (21.0%). The best observed value was 6.7 (at rugosity cm = 10, shell depth cm = 40, elevation m = 0).

Recommended Next Steps

  • Run confirmation experiments at the predicted optimal settings to validate the model.
  • Consider whether any fixed factors should be varied in a future study.

Step-by-Step Workflow

1

Preview the design

Terminal
$ python doe.py info --config use_cases/257_oyster_reef/config.json
2

Generate the runner script

Terminal
$ python doe.py generate --config use_cases/257_oyster_reef/config.json \ --output use_cases/257_oyster_reef/results/run.sh --seed 42
3

Execute the experiments

Terminal
$ bash use_cases/257_oyster_reef/results/run.sh
4

Analyze results

Terminal
$ python doe.py analyze --config use_cases/257_oyster_reef/config.json
5

Get optimization recommendations

Terminal
$ python doe.py optimize --config use_cases/257_oyster_reef/config.json
6

Multi-objective optimization

With 2 competing responses, use --multi to find the best compromise via Derringer–Suich desirability.

Terminal
$ python doe.py optimize --config use_cases/257_oyster_reef/config.json --multi
7

Generate the HTML report

Terminal
$ python doe.py report --config use_cases/257_oyster_reef/config.json \ --output use_cases/257_oyster_reef/results/report.html
