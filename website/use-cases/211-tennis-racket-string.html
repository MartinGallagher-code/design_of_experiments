Box-Behnken Design

Tennis Racket String Setup

Box-Behnken design to maximize power and control by tuning main string tension, cross string tension, and string gauge

Summary

This experiment investigates tennis racket string setup. Box-Behnken design to maximize power and control by tuning main string tension, cross string tension, and string gauge.

The design varies 3 factors: main tension kg (kg), ranging from 20 to 28, cross tension kg (kg), ranging from 18 to 26, and gauge mm (mm), ranging from 1.15 to 1.35. The goal is to optimize 2 responses: power score (pts) (maximize) and control score (pts) (maximize). Fixed conditions held constant across all runs include racket = midplus_100, string material = polyester.

A Box-Behnken design was chosen because it efficiently fits quadratic models with 3 continuous factors while avoiding extreme corner combinations — requiring only 15 runs instead of the 8 needed for a full factorial at two levels.

Quadratic response surface models were fitted to capture potential curvature and factor interactions. The RSM contour plots below visualize how pairs of factors jointly affect each response.

Key Findings

For power score, the most influential factors were main tension kg (37.9%), gauge mm (32.3%), cross tension kg (29.8%). The best observed value was 8.2 (at main tension kg = 20, cross tension kg = 26, gauge mm = 1.25).

For control score, the most influential factors were main tension kg (57.2%), gauge mm (26.2%), cross tension kg (16.6%). The best observed value was 6.4 (at main tension kg = 24, cross tension kg = 22, gauge mm = 1.25).

Recommended Next Steps

  • Run confirmation experiments at the predicted optimal settings to validate the model.
  • Consider whether any fixed factors should be varied in a future study.

Step-by-Step Workflow

1

Preview the design

Terminal
$ python doe.py info --config use_cases/211_tennis_racket_string/config.json
2

Generate the runner script

Terminal
$ python doe.py generate --config use_cases/211_tennis_racket_string/config.json \ --output use_cases/211_tennis_racket_string/results/run.sh --seed 42
3

Execute the experiments

Terminal
$ bash use_cases/211_tennis_racket_string/results/run.sh
4

Analyze results

Terminal
$ python doe.py analyze --config use_cases/211_tennis_racket_string/config.json
5

Get optimization recommendations

Terminal
$ python doe.py optimize --config use_cases/211_tennis_racket_string/config.json
6

Multi-objective optimization

With 2 competing responses, use --multi to find the best compromise via Derringer–Suich desirability.

Terminal
$ python doe.py optimize --config use_cases/211_tennis_racket_string/config.json --multi
7

Generate the HTML report

Terminal
$ python doe.py report --config use_cases/211_tennis_racket_string/config.json \ --output use_cases/211_tennis_racket_string/results/report.html
