Box-Behnken Design

Chromatography Separation

Box-Behnken design to maximize resolution and minimize run time by tuning flow rate, mobile phase ratio, and column temperature

Summary

This experiment investigates chromatography separation. Box-Behnken design to maximize resolution and minimize run time by tuning flow rate, mobile phase ratio, and column temperature.

The design varies 3 factors: flow ml min (mL/min), ranging from 0.5 to 2.0, organic pct (%), ranging from 30 to 80, and column temp c (C), ranging from 25 to 55. The goal is to optimize 2 responses: resolution (Rs) (maximize) and run time min (min) (minimize). Fixed conditions held constant across all runs include column = C18_150mm, detector = UV_254nm.

A Box-Behnken design was chosen because it efficiently fits quadratic models with 3 continuous factors while avoiding extreme corner combinations — requiring only 15 runs instead of the 8 needed for a full factorial at two levels.

Quadratic response surface models were fitted to capture potential curvature and factor interactions. The RSM contour plots below visualize how pairs of factors jointly affect each response.

Key Findings

For resolution, the most influential factors were column temp c (39.0%), organic pct (37.7%), flow ml min (23.2%). The best observed value was 3.37 (at flow ml min = 2, organic pct = 30, column temp c = 40).

For run time min, the most influential factors were column temp c (54.8%), organic pct (23.6%), flow ml min (21.6%). The best observed value was 9.3 (at flow ml min = 0.5, organic pct = 55, column temp c = 55).

Recommended Next Steps

  • Run confirmation experiments at the predicted optimal settings to validate the model.
  • Consider whether any fixed factors should be varied in a future study.

Step-by-Step Workflow

1

Preview the design

Terminal
$ python doe.py info --config use_cases/189_chromatography_separation/config.json
2

Generate the runner script

Terminal
$ python doe.py generate --config use_cases/189_chromatography_separation/config.json \ --output use_cases/189_chromatography_separation/results/run.sh --seed 42
3

Execute the experiments

Terminal
$ bash use_cases/189_chromatography_separation/results/run.sh
4

Analyze results

Terminal
$ python doe.py analyze --config use_cases/189_chromatography_separation/config.json
5

Get optimization recommendations

Terminal
$ python doe.py optimize --config use_cases/189_chromatography_separation/config.json
6

Multi-objective optimization

With 2 competing responses, use --multi to find the best compromise via Derringer–Suich desirability.

Terminal
$ python doe.py optimize --config use_cases/189_chromatography_separation/config.json --multi
7

Generate the HTML report

Terminal
$ python doe.py report --config use_cases/189_chromatography_separation/config.json \ --output use_cases/189_chromatography_separation/results/report.html
