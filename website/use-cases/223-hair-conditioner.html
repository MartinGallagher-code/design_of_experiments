Box-Behnken Design

Hair Conditioner Formulation

Box-Behnken design to maximize detangling and shine by tuning cetyl alcohol, silicone dimethicone, and protein content

Summary

This experiment investigates hair conditioner formulation. Box-Behnken design to maximize detangling and shine by tuning cetyl alcohol, silicone dimethicone, and protein content.

The design varies 3 factors: cetyl pct (%), ranging from 2 to 6, dimethicone pct (%), ranging from 0.5 to 3.0, and protein pct (%), ranging from 0.5 to 2.0. The goal is to optimize 2 responses: detangle score (pts) (maximize) and shine score (pts) (maximize). Fixed conditions held constant across all runs include base ph = 4.5, preservative = standard.

A Box-Behnken design was chosen because it efficiently fits quadratic models with 3 continuous factors while avoiding extreme corner combinations — requiring only 15 runs instead of the 8 needed for a full factorial at two levels.

Quadratic response surface models were fitted to capture potential curvature and factor interactions. The RSM contour plots below visualize how pairs of factors jointly affect each response.

Key Findings

For detangle score, the most influential factors were cetyl pct (68.1%), protein pct (21.9%), dimethicone pct (10.0%). The best observed value was 7.1 (at cetyl pct = 4, dimethicone pct = 3, protein pct = 0.5).

For shine score, the most influential factors were cetyl pct (51.8%), protein pct (30.7%), dimethicone pct (17.5%). The best observed value was 8.0 (at cetyl pct = 2, dimethicone pct = 1.75, protein pct = 0.5).

Recommended Next Steps

  • Run confirmation experiments at the predicted optimal settings to validate the model.
  • Consider whether any fixed factors should be varied in a future study.

Step-by-Step Workflow

1

Preview the design

Terminal
$ python doe.py info --config use_cases/223_hair_conditioner/config.json
2

Generate the runner script

Terminal
$ python doe.py generate --config use_cases/223_hair_conditioner/config.json \ --output use_cases/223_hair_conditioner/results/run.sh --seed 42
3

Execute the experiments

Terminal
$ bash use_cases/223_hair_conditioner/results/run.sh
4

Analyze results

Terminal
$ python doe.py analyze --config use_cases/223_hair_conditioner/config.json
5

Get optimization recommendations

Terminal
$ python doe.py optimize --config use_cases/223_hair_conditioner/config.json
6

Multi-objective optimization

With 2 competing responses, use --multi to find the best compromise via Derringer–Suich desirability.

Terminal
$ python doe.py optimize --config use_cases/223_hair_conditioner/config.json --multi
7

Generate the HTML report

Terminal
$ python doe.py report --config use_cases/223_hair_conditioner/config.json \ --output use_cases/223_hair_conditioner/results/report.html
