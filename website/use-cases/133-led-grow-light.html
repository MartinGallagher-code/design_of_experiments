Box-Behnken Design

LED Grow Light Spectrum

Box-Behnken design to maximize photosynthesis rate and minimize heat output by tuning red-to-blue ratio, intensity, and photoperiod

Summary

This experiment investigates led grow light spectrum. Box-Behnken design to maximize photosynthesis rate and minimize heat output by tuning red-to-blue ratio, intensity, and photoperiod.

The design varies 3 factors: red blue ratio (ratio), ranging from 1 to 8, ppfd (umol/m2/s), ranging from 200 to 800, and photoperiod hrs (hrs), ranging from 12 to 20. The goal is to optimize 2 responses: growth rate g (g/day) (maximize) and heat watts (W) (minimize). Fixed conditions held constant across all runs include plant type = leafy_greens, distance cm = 30.

A Box-Behnken design was chosen because it efficiently fits quadratic models with 3 continuous factors while avoiding extreme corner combinations — requiring only 15 runs instead of the 8 needed for a full factorial at two levels.

Quadratic response surface models were fitted to capture potential curvature and factor interactions. The RSM contour plots below visualize how pairs of factors jointly affect each response.

Key Findings

For growth rate g, the most influential factors were red blue ratio (54.4%), photoperiod hrs (29.3%), ppfd (16.3%). The best observed value was 7.2 (at red blue ratio = 1, ppfd = 800, photoperiod hrs = 16).

For heat watts, the most influential factors were red blue ratio (59.5%), photoperiod hrs (26.5%), ppfd (14.0%). The best observed value was 20.0 (at red blue ratio = 8, ppfd = 500, photoperiod hrs = 12).

Recommended Next Steps

  • Run confirmation experiments at the predicted optimal settings to validate the model.
  • Consider whether any fixed factors should be varied in a future study.

Step-by-Step Workflow

1

Preview the design

Terminal
$ python doe.py info --config use_cases/133_led_grow_light/config.json
2

Generate the runner script

Terminal
$ python doe.py generate --config use_cases/133_led_grow_light/config.json \ --output use_cases/133_led_grow_light/results/run.sh --seed 42
3

Execute the experiments

Terminal
$ bash use_cases/133_led_grow_light/results/run.sh
4

Analyze results

Terminal
$ python doe.py analyze --config use_cases/133_led_grow_light/config.json
5

Get optimization recommendations

Terminal
$ python doe.py optimize --config use_cases/133_led_grow_light/config.json
6

Multi-objective optimization

With 2 competing responses, use --multi to find the best compromise via Derringer–Suich desirability.

Terminal
$ python doe.py optimize --config use_cases/133_led_grow_light/config.json --multi
7

Generate the HTML report

Terminal
$ python doe.py report --config use_cases/133_led_grow_light/config.json \ --output use_cases/133_led_grow_light/results/report.html
