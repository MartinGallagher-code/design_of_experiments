Box-Behnken

Checkpoint/Restart I/O Optimization

Optimize checkpoint write throughput while minimizing application disruption on burst-buffer-equipped HPC systems.

How to Run

terminal
$ python doe.py info --config use_cases/19_checkpoint_restart/config.json $ python doe.py generate --config use_cases/19_checkpoint_restart/config.json --output results/run.sh --seed 42 $ bash results/run.sh $ python doe.py analyze --config use_cases/19_checkpoint_restart/config.json $ python doe.py optimize --config use_cases/19_checkpoint_restart/config.json $ python doe.py report --config use_cases/19_checkpoint_restart/config.json --output report.html
