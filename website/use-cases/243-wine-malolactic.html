Box-Behnken Design

Wine Malolactic Fermentation

Box-Behnken design to maximize MLF completion and minimize VA by tuning inoculation rate, cellar temperature, and free SO2 level

Summary

This experiment investigates wine malolactic fermentation. Box-Behnken design to maximize MLF completion and minimize VA by tuning inoculation rate, cellar temperature, and free SO2 level.

The design varies 3 factors: inoc rate (g/hL), ranging from 0.5 to 2.0, cellar temp c (C), ranging from 15 to 22, and free so2 ppm (ppm), ranging from 5 to 25. The goal is to optimize 2 responses: mlf completion pct (%) (maximize) and va g L (g/L) (minimize). Fixed conditions held constant across all runs include wine type = chardonnay, ph = 3.4.

A Box-Behnken design was chosen because it efficiently fits quadratic models with 3 continuous factors while avoiding extreme corner combinations — requiring only 15 runs instead of the 8 needed for a full factorial at two levels.

Quadratic response surface models were fitted to capture potential curvature and factor interactions. The RSM contour plots below visualize how pairs of factors jointly affect each response.

Key Findings

For mlf completion pct, the most influential factors were free so2 ppm (43.2%), inoc rate (33.5%), cellar temp c (23.3%). The best observed value was 97.0 (at inoc rate = 1.25, cellar temp c = 22, free so2 ppm = 25).

For va g L, the most influential factors were inoc rate (51.9%), free so2 ppm (35.5%), cellar temp c (12.5%). The best observed value was 0.21 (at inoc rate = 1.25, cellar temp c = 15, free so2 ppm = 25).

Recommended Next Steps

  • Run confirmation experiments at the predicted optimal settings to validate the model.
  • Consider whether any fixed factors should be varied in a future study.

Step-by-Step Workflow

1

Preview the design

Terminal
$ python doe.py info --config use_cases/243_wine_malolactic/config.json
2

Generate the runner script

Terminal
$ python doe.py generate --config use_cases/243_wine_malolactic/config.json \ --output use_cases/243_wine_malolactic/results/run.sh --seed 42
3

Execute the experiments

Terminal
$ bash use_cases/243_wine_malolactic/results/run.sh
4

Analyze results

Terminal
$ python doe.py analyze --config use_cases/243_wine_malolactic/config.json
5

Get optimization recommendations

Terminal
$ python doe.py optimize --config use_cases/243_wine_malolactic/config.json
6

Multi-objective optimization

With 2 competing responses, use --multi to find the best compromise via Derringer–Suich desirability.

Terminal
$ python doe.py optimize --config use_cases/243_wine_malolactic/config.json --multi
7

Generate the HTML report

Terminal
$ python doe.py report --config use_cases/243_wine_malolactic/config.json \ --output use_cases/243_wine_malolactic/results/report.html
