Central Composite Design

Encaustic Wax Painting

Central composite design to maximize adhesion and minimize cracking by tuning wax temperature, damar resin ratio, and fuse heat gun distance

Summary

This experiment investigates encaustic wax painting. Central composite design to maximize adhesion and minimize cracking by tuning wax temperature, damar resin ratio, and fuse heat gun distance.

The design varies 3 factors: wax temp c (C), ranging from 80 to 120, damar pct (%), ranging from 5 to 25, and fuse dist cm (cm), ranging from 5 to 20. The goal is to optimize 2 responses: adhesion score (pts) (maximize) and cracking score (pts) (minimize). Fixed conditions held constant across all runs include wax = purified_beeswax, support = birch_panel.

A Central Composite Design (CCD) was selected to fit a full quadratic response surface model, including curvature and interaction effects. With 3 factors this produces 22 runs including center points and axial (star) points that extend beyond the factorial range.

Quadratic response surface models were fitted to capture potential curvature and factor interactions. The RSM contour plots below visualize how pairs of factors jointly affect each response.

Key Findings

For adhesion score, the most influential factors were fuse dist cm (42.0%), wax temp c (38.1%), damar pct (19.8%). The best observed value was 7.8 (at wax temp c = 100, damar pct = 15, fuse dist cm = 12.5).

For cracking score, the most influential factors were wax temp c (47.2%), fuse dist cm (29.2%), damar pct (23.6%). The best observed value was 2.2 (at wax temp c = 100, damar pct = 15, fuse dist cm = 12.5).

Recommended Next Steps

  • Run confirmation experiments at the predicted optimal settings to validate the model.
  • Consider whether any fixed factors should be varied in a future study.

Step-by-Step Workflow

1

Preview the design

Terminal
$ python doe.py info --config use_cases/290_encaustic_wax/config.json
2

Generate the runner script

Terminal
$ python doe.py generate --config use_cases/290_encaustic_wax/config.json \ --output use_cases/290_encaustic_wax/results/run.sh --seed 42
3

Execute the experiments

Terminal
$ bash use_cases/290_encaustic_wax/results/run.sh
4

Analyze results

Terminal
$ python doe.py analyze --config use_cases/290_encaustic_wax/config.json
5

Get optimization recommendations

Terminal
$ python doe.py optimize --config use_cases/290_encaustic_wax/config.json
6

Multi-objective optimization

With 2 competing responses, use --multi to find the best compromise via Derringer–Suich desirability.

Terminal
$ python doe.py optimize --config use_cases/290_encaustic_wax/config.json --multi
7

Generate the HTML report

Terminal
$ python doe.py report --config use_cases/290_encaustic_wax/config.json \ --output use_cases/290_encaustic_wax/results/report.html
