Box-Behnken Design

Gel Electrophoresis Resolution

Box-Behnken design to maximize band resolution and minimize run time by tuning gel percentage, voltage, and loading volume

Summary

This experiment investigates gel electrophoresis resolution. Box-Behnken design to maximize band resolution and minimize run time by tuning gel percentage, voltage, and loading volume.

The design varies 3 factors: gel pct (%), ranging from 0.8 to 2.5, voltage v cm (V/cm), ranging from 3 to 10, and load ul (uL), ranging from 5 to 25. The goal is to optimize 2 responses: resolution (pts) (maximize) and run time min (min) (minimize). Fixed conditions held constant across all runs include buffer = TAE_1x, stain = ethidium_bromide.

A Box-Behnken design was chosen because it efficiently fits quadratic models with 3 continuous factors while avoiding extreme corner combinations — requiring only 15 runs instead of the 8 needed for a full factorial at two levels.

Quadratic response surface models were fitted to capture potential curvature and factor interactions. The RSM contour plots below visualize how pairs of factors jointly affect each response.

Key Findings

For resolution, the most influential factors were voltage v cm (36.8%), load ul (32.0%), gel pct (31.2%). The best observed value was 7.8 (at gel pct = 0.8, voltage v cm = 6.5, load ul = 5).

For run time min, the most influential factors were gel pct (37.3%), load ul (36.2%), voltage v cm (26.4%). The best observed value was 24.0 (at gel pct = 1.65, voltage v cm = 6.5, load ul = 15).

Recommended Next Steps

  • Run confirmation experiments at the predicted optimal settings to validate the model.
  • Consider whether any fixed factors should be varied in a future study.

Step-by-Step Workflow

1

Preview the design

Terminal
$ python doe.py info --config use_cases/196_gel_electrophoresis/config.json
2

Generate the runner script

Terminal
$ python doe.py generate --config use_cases/196_gel_electrophoresis/config.json \ --output use_cases/196_gel_electrophoresis/results/run.sh --seed 42
3

Execute the experiments

Terminal
$ bash use_cases/196_gel_electrophoresis/results/run.sh
4

Analyze results

Terminal
$ python doe.py analyze --config use_cases/196_gel_electrophoresis/config.json
5

Get optimization recommendations

Terminal
$ python doe.py optimize --config use_cases/196_gel_electrophoresis/config.json
6

Multi-objective optimization

With 2 competing responses, use --multi to find the best compromise via Derringer–Suich desirability.

Terminal
$ python doe.py optimize --config use_cases/196_gel_electrophoresis/config.json --multi
7

Generate the HTML report

Terminal
$ python doe.py report --config use_cases/196_gel_electrophoresis/config.json \ --output use_cases/196_gel_electrophoresis/results/report.html
