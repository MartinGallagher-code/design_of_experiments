Box-Behnken Design

Toothpaste Cleaning Power

Box-Behnken design to maximize cleaning efficacy and minimize abrasion by tuning silica abrasive content, fluoride concentration, and foam agent level

Summary

This experiment investigates toothpaste cleaning power. Box-Behnken design to maximize cleaning efficacy and minimize abrasion by tuning silica abrasive content, fluoride concentration, and foam agent level.

The design varies 3 factors: silica pct (%), ranging from 10 to 25, fluoride ppm (ppm), ranging from 500 to 1500, and sls pct (%), ranging from 0.5 to 2.5. The goal is to optimize 2 responses: cleaning score (pts) (maximize) and rda index (RDA) (minimize). Fixed conditions held constant across all runs include flavor = mint, ph = 7.0.

A Box-Behnken design was chosen because it efficiently fits quadratic models with 3 continuous factors while avoiding extreme corner combinations — requiring only 15 runs instead of the 8 needed for a full factorial at two levels.

Quadratic response surface models were fitted to capture potential curvature and factor interactions. The RSM contour plots below visualize how pairs of factors jointly affect each response.

Key Findings

For cleaning score, the most influential factors were fluoride ppm (40.5%), sls pct (34.8%), silica pct (24.7%). The best observed value was 7.6 (at silica pct = 10, fluoride ppm = 1500, sls pct = 1.5).

For rda index, the most influential factors were fluoride ppm (45.3%), sls pct (28.4%), silica pct (26.3%). The best observed value was 49.0 (at silica pct = 17.5, fluoride ppm = 1000, sls pct = 1.5).

Recommended Next Steps

  • Run confirmation experiments at the predicted optimal settings to validate the model.
  • Consider whether any fixed factors should be varied in a future study.

Step-by-Step Workflow

1

Preview the design

Terminal
$ python doe.py info --config use_cases/225_toothpaste_formulation/config.json
2

Generate the runner script

Terminal
$ python doe.py generate --config use_cases/225_toothpaste_formulation/config.json \ --output use_cases/225_toothpaste_formulation/results/run.sh --seed 42
3

Execute the experiments

Terminal
$ bash use_cases/225_toothpaste_formulation/results/run.sh
4

Analyze results

Terminal
$ python doe.py analyze --config use_cases/225_toothpaste_formulation/config.json
5

Get optimization recommendations

Terminal
$ python doe.py optimize --config use_cases/225_toothpaste_formulation/config.json
6

Multi-objective optimization

With 2 competing responses, use --multi to find the best compromise via Derringer–Suich desirability.

Terminal
$ python doe.py optimize --config use_cases/225_toothpaste_formulation/config.json --multi
7

Generate the HTML report

Terminal
$ python doe.py report --config use_cases/225_toothpaste_formulation/config.json \ --output use_cases/225_toothpaste_formulation/results/report.html
