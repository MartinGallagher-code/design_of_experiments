Box-Behnken Design

DIY Antenna Tuning

Box-Behnken design to maximize signal strength and minimize SWR by tuning element length, height above ground, and feed point impedance

Summary

This experiment investigates diy antenna tuning. Box-Behnken design to maximize signal strength and minimize SWR by tuning element length, height above ground, and feed point impedance.

The design varies 3 factors: element pct (%), ranging from 90 to 110, height m (m), ranging from 3 to 10, and feedpoint ohm (ohm), ranging from 25 to 75. The goal is to optimize 2 responses: gain dbi (dBi) (maximize) and swr (ratio) (minimize). Fixed conditions held constant across all runs include band = 2m_VHF, type = ground_plane.

A Box-Behnken design was chosen because it efficiently fits quadratic models with 3 continuous factors while avoiding extreme corner combinations — requiring only 15 runs instead of the 8 needed for a full factorial at two levels.

Quadratic response surface models were fitted to capture potential curvature and factor interactions. The RSM contour plots below visualize how pairs of factors jointly affect each response.

Key Findings

For gain dbi, the most influential factors were element pct (55.3%), height m (26.8%), feedpoint ohm (18.0%). The best observed value was 5.3 (at element pct = 90, height m = 6.5, feedpoint ohm = 75).

For swr, the most influential factors were height m (58.0%), element pct (39.6%), feedpoint ohm (2.4%). The best observed value was 1.08 (at element pct = 100, height m = 10, feedpoint ohm = 75).

Recommended Next Steps

  • Run confirmation experiments at the predicted optimal settings to validate the model.
  • Consider whether any fixed factors should be varied in a future study.

Step-by-Step Workflow

1

Preview the design

Terminal
$ python doe.py info --config use_cases/273_antenna_tuning/config.json
2

Generate the runner script

Terminal
$ python doe.py generate --config use_cases/273_antenna_tuning/config.json \ --output use_cases/273_antenna_tuning/results/run.sh --seed 42
3

Execute the experiments

Terminal
$ bash use_cases/273_antenna_tuning/results/run.sh
4

Analyze results

Terminal
$ python doe.py analyze --config use_cases/273_antenna_tuning/config.json
5

Get optimization recommendations

Terminal
$ python doe.py optimize --config use_cases/273_antenna_tuning/config.json
6

Multi-objective optimization

With 2 competing responses, use --multi to find the best compromise via Derringer–Suich desirability.

Terminal
$ python doe.py optimize --config use_cases/273_antenna_tuning/config.json --multi
7

Generate the HTML report

Terminal
$ python doe.py report --config use_cases/273_antenna_tuning/config.json \ --output use_cases/273_antenna_tuning/results/report.html
