Box-Behnken Design

Candle Making Optimization

Box-Behnken design to maximize burn time and scent throw by tuning wick size, fragrance load, and pour temperature

Summary

This experiment investigates candle making optimization. Box-Behnken design to maximize burn time and scent throw by tuning wick size, fragrance load, and pour temperature.

The design varies 3 factors: wick size (mm), ranging from 4 to 10, fragrance pct (%), ranging from 4 to 12, and pour temp c (C), ranging from 55 to 80. The goal is to optimize 2 responses: burn hrs (hrs) (maximize) and scent throw (pts) (maximize). Fixed conditions held constant across all runs include wax type = soy, container = 8oz_jar.

A Box-Behnken design was chosen because it efficiently fits quadratic models with 3 continuous factors while avoiding extreme corner combinations — requiring only 15 runs instead of the 8 needed for a full factorial at two levels.

Quadratic response surface models were fitted to capture potential curvature and factor interactions. The RSM contour plots below visualize how pairs of factors jointly affect each response.

Key Findings

For burn hrs, the most influential factors were pour temp c (70.3%), fragrance pct (21.1%), wick size (8.6%). The best observed value was 51.0 (at wick size = 10, fragrance pct = 8, pour temp c = 80).

For scent throw, the most influential factors were fragrance pct (44.0%), pour temp c (28.5%), wick size (27.5%). The best observed value was 7.7 (at wick size = 4, fragrance pct = 8, pour temp c = 80).

Recommended Next Steps

  • Run confirmation experiments at the predicted optimal settings to validate the model.
  • Consider whether any fixed factors should be varied in a future study.

Step-by-Step Workflow

1

Preview the design

Terminal
$ python doe.py info --config use_cases/140_candle_making/config.json
2

Generate the runner script

Terminal
$ python doe.py generate --config use_cases/140_candle_making/config.json \ --output use_cases/140_candle_making/results/run.sh --seed 42
3

Execute the experiments

Terminal
$ bash use_cases/140_candle_making/results/run.sh
4

Analyze results

Terminal
$ python doe.py analyze --config use_cases/140_candle_making/config.json
5

Get optimization recommendations

Terminal
$ python doe.py optimize --config use_cases/140_candle_making/config.json
6

Multi-objective optimization

With 2 competing responses, use --multi to find the best compromise via Derringer–Suich desirability.

Terminal
$ python doe.py optimize --config use_cases/140_candle_making/config.json --multi
7

Generate the HTML report

Terminal
$ python doe.py report --config use_cases/140_candle_making/config.json \ --output use_cases/140_candle_making/results/report.html
